Esercizio 9: Estensioni - JSON Enc

Obiettivo

Usa l'estensione json-enc per inviare dati come JSON invece che form-urlencoded. Crea un pulsante che invia un oggetto JSON {"value": "test"} a /api/val.

Attributi:

  • hx-ext="json-enc" - abilita encoding JSON
  • hx-vals='{"value": "test"}' - valori da inviare come JSON
  • hx-post="/api/val" - metodo POST
