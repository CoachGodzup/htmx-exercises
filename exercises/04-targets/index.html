Esercizio 4: Target Selection

Obiettivo

Usa selettori CSS avanzati per hx-target. Crea un pulsante che mette la risposta in un elemento diverso da quello che ha l'attributo hx-get.

Target comuni:

  • hx-target="#id" - per ID
  • hx-target=".class" - per classe
  • hx-target="div" - per tag
  • hx-target="closest div" - l'antenato più vicino
  • hx-target="next .result" - l'elemento successivo con classe
  • hx-target="previous .result" - l'elemento precedente
