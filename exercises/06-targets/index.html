Esercizio 6: Target Selection Avanzato

Obiettivo

Usa selettori CSS avanzati per hx-target. Fai in modo che la risposta vada nell'elemento con id target-result che si trova dentro un container.

Target utili:

  • hx-target="#target-result" - per ID
  • hx-target=".result" - per classe
  • hx-target="closest .container" - elemento antenato più vicino
  • hx-target="next .result" - elemento successivo
  • hx-target="previous .result" - elemento precedente
